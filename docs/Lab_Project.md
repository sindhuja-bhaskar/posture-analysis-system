# Lab Project: Real-Time Posture Analysis System

This project develops a real-time posture analysis system using a camera and computer vision techniques. The system captures live video through a webcam and processes it using OpenCV along with MediaPipe to detect human body landmarks. Based on the positions of key points such as the neck, shoulders, and hips, the system calculates body angles and evaluates the user's posture.

Using Object-Oriented Programming (OOP) principles, the project is designed with modular components for video capture, pose detection, and posture analysis. The system classifies posture as good or bad and displays feedback in real time. This helps in promoting healthy sitting and standing habits and can be extended for fitness monitoring or workplace ergonomics.
